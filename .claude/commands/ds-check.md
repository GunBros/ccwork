다음 파일들을 읽어라:

- docs/design/rules.md
- docs/design/tokens.md

그런 다음 **$ARGUMENTS** (파일 경로 또는 현재 편집 중인 파일)를 읽고,
스타일 코드가 디자인 시스템 Do/Don't 규칙을 준수하는지 검토해줘.

검토 기준:

1. 순수 검정(`#000000`, `text-black`) 사용 여부
2. 일반 shadow 클래스(`shadow-sm/md/lg/xl`) 사용 여부
3. 섹션/사이드바 구분에 border 사용 여부
4. 리스트 구분선(`divide-*`) 사용 여부
5. 인라인 `box-shadow` 직접 사용 여부
6. 하드코딩된 색상값 (토큰 대신 임의 HEX) 사용 여부

출력 형식:

- 위반 항목: 코드 위치 + 위반 규칙 + 권장 대안
- 준수 항목: 잘 적용된 패턴 요약
- 종합 평가: 개선 우선순위
