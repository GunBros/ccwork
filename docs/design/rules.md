# Do's & Don'ts — 작업 전 체크리스트

> 이 파일은 `/ds-check` 스킬과 `ds-lint` 훅의 판단 기준이다.
> 스타일 작업 완료 후 이 목록을 기준으로 검토한다.

---

## 색상

|           | 규칙                                                                    |
| --------- | ----------------------------------------------------------------------- |
| **Do**    | 본문 텍스트에 `on_surface` (#2b3437) 사용                               |
| **Do**    | 장문 가독성이 필요한 곳에 `on_surface_variant` (#586064) 사용           |
| **Do**    | 선택 상태(사이드바 아이템)에 `surface_container_highest` (#dbe4e7) 사용 |
| **Do**    | accent 색상(`tertiary` #0053dc)은 의도적 액션(버튼, 포커스)에만 사용    |
| **Don't** | 텍스트에 순수 검정 `#000000` / `text-black` 사용 금지                   |
| **Don't** | `tertiary`를 장식 목적으로 남용 금지                                    |

---

## 경계 & 구분

|           | 규칙                                                                              |
| --------- | --------------------------------------------------------------------------------- |
| **Do**    | 영역 구분은 배경색 전환으로 표현                                                  |
| **Do**    | 접근성 문제가 있을 때만 Ghost Border (`outline_variant` 15% opacity) 사용         |
| **Don't** | 섹션/사이드바 구분에 `1px solid border` 사용 금지                                 |
| **Don't** | 리스트 아이템 사이에 `divide-y`, `divide-x` 사용 금지 → `spacing.4` 간격으로 대체 |

---

## 엘리베이션

|           | 규칙                                                                                    |
| --------- | --------------------------------------------------------------------------------------- |
| **Do**    | 카드 깊이는 Tonal Layering으로 표현 (`surface_container_lowest` on `surface_container`) |
| **Do**    | 실제 플로팅 요소(모달, 팝오버)에만 Ambient Shadow 적용                                  |
| **Don't** | 일반 카드에 `shadow-sm`, `shadow-md`, `shadow-lg`, `shadow-xl` 사용 금지                |
| **Don't** | 기본 `box-shadow` CSS 사용 금지                                                         |

---

## 타이포그래피

|           | 규칙                                                             |
| --------- | ---------------------------------------------------------------- |
| **Do**    | 메타데이터 레이블은 `uppercase` + `letter-spacing: +0.05em` 적용 |
| **Do**    | TIL 제목에 `headline-md` (1.75rem) + `line-height: 1.4` 적용     |
| **Don't** | `display-lg` (3.5rem)를 랜딩 이외 위치에 사용 금지               |

---

## 아이콘 & 여백

|           | 규칙                                                                   |
| --------- | ---------------------------------------------------------------------- |
| **Do**    | 여백을 구조적 요소로 사용 — 부족하다면 늘릴 것                         |
| **Do**    | 레이아웃 블록 간 여백은 `spacing.10` (3.5rem) 이상 유지                |
| **Don't** | 장식 목적의 아이콘 사용 금지 — 즉각적인 기능적 명확성이 있을 때만 허용 |
| **Don't** | `spacing.10` 대형 여백을 줄여 '숨막히는' 레이아웃 만들기 금지          |

---

## ds-lint 자동 감지 패턴

훅이 자동으로 감지하는 위반 코드:

| 코드   | 감지 패턴                                          | 해당 규칙                |
| ------ | -------------------------------------------------- | ------------------------ |
| DS-001 | `text-black`, `#000000`                            | 순수 검정 금지           |
| DS-002 | `shadow-sm`, `shadow-md`, `shadow-lg`, `shadow-xl` | 일반 shadow 금지         |
| DS-003 | `divide-y`, `divide-x`, `divide-`                  | 리스트 구분선 금지       |
| DS-004 | 인라인 `border:.*solid`                            | 명시적 solid border 금지 |
